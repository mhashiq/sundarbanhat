# সুন্দরবন হাট (Sundarban Hat)

"সুন্দরবনের খাঁটি খাবার, সরাসরি আপনার ঘরে"

এটি একটি সম্পূর্ণ স্ট্যাটিক, মোবাইল-ফার্স্ট ওয়েবসাইট যা সুন্দরবনের প্রাকৃতিক ও ঐতিহ্যবাহী খাদ্যপণ্য প্রদর্শনের জন্য তৈরি করা হয়েছে। 

## প্রযুক্তি
- HTML5
- CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)

## কিভাবে রান করবেন (VS Code)

১. VS Code এ ফোল্ডারটি খুলুন।
২. Extensions এ গিয়ে **Live Server** (Ritwick Dey) ইনস্টল করুন।
৩. `index.html` ফাইলটিতে রাইট-ক্লিক করুন এবং **"Open with Live Server"** সিলেক্ট করুন।
৪. আপনার ব্রাউজারে সাইটটি ওপেন হবে এবং লোকালহোস্টে চলবে।

## ফোল্ডার স্ট্রাকচার
- `assets/css/` - স্টাইলশীট (`style.css`)
- `assets/js/` - জাভাস্ক্রিপ্ট (`main.js`)
- `assets/images/` - ওয়েবসাইটের ছবি (এখানে লোগো এবং পণ্যের ছবি রাখুন)

## ফাইলসমূহ
- `index.html` - হোমপেজ
- `products.html` - পণ্যসমূহ
- `about.html` - সুন্দরবনের গল্প ও আমাদের সম্পর্কে
- `why-us.html` - কেন সুন্দরবন হাট
- `contact.html` - যোগাযোগ
